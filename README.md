# Travel Booking App 

## Project Description
This is a simple Travel Booking Application built using Python and Streamlit. It helps users login, book their travel tickets, and view their booking history in an organized way.

- Easy Ticket Booking Form
- View saved Booking History
- Responsive Mobile Layout

## Tech Stack
- **Language:** Python
- **Framework:** Streamlit
- **Data Storage:** JSON (in Data/ folder)
- **Deployment:** Streamlit Cloud

## Setup Instructions
1. Clone this repository to your local machine.
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
