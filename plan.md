# Travel Booking Application - Project Plan

## Core Features to Build
1. **Destination Explorer:** Users can view tourist places (Ooty, Goa, etc.) along with prices.
2. **Search & Filter:** Filter destinations based on price and ratings.
3. **Booking System:** A simple module to check availability and book a trip.

## Technology Stack
* Frontend/UI: HTML, CSS, JavaScript (Mobile Friendly)
* Backend/Logic: Python (Data management)
